Installé SteamCMD

Téléchargé steamcmd.zip sur https://steamcdn-a.akamaihd.net/client/installer/steamcmd.zip
Extraire le fichier "steamcmd.zip" ou bon vous semble.

Créez un compte utilisateur nommé steam pour exécuter SteamCMD en toute sécurité, en l'isolant du reste du système d'exploitation.
Avertissement: Ne pas exécuter steamcmd en tant qu'utilisateur root. Cela représente un risque pour la sécurité.

1. En tant qu'utilisateur root, créez l' utilisateur steam :

                sudo useradd -m steam
                sudo passwd steam

2. Allez dans son dossier "home": 

                sudo -u steam -s
                cd /home/steam

3. Il est recommandé d'installer le package SteamCMD à partir de vos référentiels de distribution, s'ils sont disponibles : 

sudo apt install steamcmd

Noter: Si vous utilisez une machine 64 bits, vous devrez ajouter multiverse :

                sudo add-apt-repository multiverse
                sudo dpkg --add-architecture i386
                sudo apt update
                sudo apt install lib32gcc-s1 steamcmd
